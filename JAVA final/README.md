#  Proyecto Final - Construcción de Software I

Aplicación de escritorio desarrollada en **Java Swing** como parte del curso **Construcción de Software I**.  
Esta versión está completamente funcional y probada en **Visual Studio Code** con el **Extension Pack for Java**.



##  Objetivo del Proyecto

Aplicar los conceptos de programación en Java para construir interfaces gráficas utilizando **Java Swing**  
y demostrar el uso de componentes GUI fundamentales y control de versiones con **GitHub**.



##  Descripción de la Aplicación

La aplicación simula un sistema de gestión simple con un login inicial y una interfaz principal donde se pueden agregar usuarios a una tabla.
